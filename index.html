Скачивайте схемы для Minecraft из litematica.json

Как это работает:

Сайт автоматически загружает схемы из файла litematica.json, который должен находиться рядом с index.html.

Формат файла должен быть такой:

{ "schemes": [ { "name": "Название схемы", "version": "Версия Minecraft", "path": "/путь/к/файлу.litematic" } ] }
Загрузка схем из litematica.json...

О файле litematica.json

Чтобы добавить свои схемы:

  1. Создайте файл litematica.json рядом с index.html
  2. Добавьте свои схемы в формате JSON как показано выше
  3. Обновите страницу или нажмите кнопку "Обновить список схем"

Примечание: Путь к файлу может быть абсолютным URL (https://...) или относительным путем
