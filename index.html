Формат файла litematica.json

Для добавления своих схем создайте файл litematica.json рядом с index.html:

{
  "schemes": [
    {
      "name": "Название схемы",
      "version": "1.19.2",
      "path": "https://ваш-сайт/scheme.litematic",
      "description": "Описание...",
      "creator": "Автор",
      "difficulty": "Легкая",
      "game_mode": "Выживание",
      "materials": ["Дуб", "Камень"],
      "size": "Средний",
      "build_time": "2-3 часа",
      "features": ["Ферма", "Склад"],
      "deleted_on_the_source": false, // удалена на исходнике
      "deleted": false // удалена везде (нельзя скачать)
    }
  ]
}

Важные пояснения:

deleted: true - схема удалена ВЕЗДЕ, скачать НЕЛЬЗЯ
deleted_on_the_source: true - удалена на исходном сайте, но у нас есть копия, скачать МОЖНО
deleted: false - активная схема, доступна для скачивания
path - ссылка для скачивания с ВАШЕГО сайта
