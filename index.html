Формат файла litematica.json

Для добавления своих схем создайте файл litematica.json рядом с index.html со следующей структурой:

{
  "schemes": [
    {
      "name": "Название схемы",
      "version": "1.19.2",
      "path": "https://example.com/scheme.litematic",
      "description": "Описание схемы...",
      "creator": "Автор",
      "finder": "Кто нашел",
      "source": "Источник",
      "difficulty": "Легкая",
      "game_mode": "Выживание",
      "materials": ["Дуб", "Камень", "Стекло"],
      "size": "Средний",
      "redstone": "Базовый",
      "build_time": "2-3 часа",
      "features": ["Автоферма", "Склад"],
      "deleted_on_the_source": false,
      "deleted": false
    }
  ]
}

Описание полей:

name: Название схемы
version: Версия Minecraft
path: Ссылка для скачивания
description: Подробное описание
creator: Автор схемы
difficulty: Сложность (Легкая/Средняя/Сложная)
game_mode: Режим игры (Выживание/Креатив)
materials: Материалы для постройки
deleted: Если true - схема в архиве
