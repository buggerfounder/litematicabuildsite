Формат файла litematica.json
{
  "schemes": [
    {
      "name": "3x3 дверная машина",
      "version": "1.16.5-1.21.1+",
      "path": "https://example.com/scheme.litematic",
      "description": "Компактная дверная машина 3x3",
      "creator": "Автор (ссылка)",
      "finder": "Кто нашел",
      "source": "Источник: ссылка",
      "difficulty": "Легкая",
      "game_mode": "Выживание",
      "materials": ["красная пыль", "поршни"],
      "size": "Малый",
      "redstone": "Средний",
      "build_time": "10-15 минут",
      "features": ["компактный", "автоматический"],
      "deleted_on_the_source": false,
      "deleted": false
    }
  ]
}

Обязательные поля: name, version, path

Рекомендуемые: description, creator, difficulty, game_mode

Опциональные: остальные поля

deleted_on_the_source: true если удалено на оригинальном сайте

deleted: true если полностью удалено (нельзя скачать)
