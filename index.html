О файле litematica.json

Чтобы добавить свои схемы вам надо:

  1. Создайте файл litematica.json на рабочем столе
  2. Напишите в файле litematica.json этот код заменяя на свое:
{ "schemes": [ { "name": "Название схемы", "version": "Версия Minecraft", "path": "https://сайт/путь/к/файлу.litematic", "description": "Описание схемы", "creator": "Создатель схемы", "finder": "Кто нашел схему", "source": "Сайт откуда скачана", "deleted_on_the_source": false, "deleted": false } ] }
Важно: deleted_on_the_source: true - схема удалена на оригинальном сайте, но доступна здесь
deleted: true - схема полностью удалена и недоступна для скачивания
  1. Потом напишите автору в дискорд cobaka1967 предварительно кинув заявки в друзья! И ожидайте пока он рассмотрит
